AppGate

Mar 21, 2017

AppGate (Application Gateway) is a VPN/network access control solution. It provides granular and secure access to applications and services hosted anywhere. I will start with the interesting bits and jump directly to the tech stack and my role in the project, followed by the architectural overview.

Tech Stack

  • Backend: No publicly disclosed info so cannot mention this part.
  • Management Console: Based on my open source project KOAN. AngularJS and Node.js.
  • Client Core: Xamarin and C#. Shared code on all supported dekstop and mobile platforms.
  • Client UI: Polymer app running in Electron on desktops, WebView on mobiles.

I have personally worked on all aspects of the project during the design and development, which took over 4 years. Starting with recruitment and architectural decisions, to the development of first 3 versions. Server side was developed as micro-services after consulting with expert opinions and our consensus. Management Console is forked from my open source project KOAN. Same goes for Android and iOS clients, which I forked from my existing WebView based projects.
